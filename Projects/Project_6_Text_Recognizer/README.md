## 📋 Project Overview
In this project, I learned how to extract text from images (like scanned documents or receipts) using Computer Vision. I used OpenCV to clean up the images and Google's Tesseract OCR to read the text.

## ⚙️ What I Learned (Computer Vision)
I found that Tesseract makes mistakes if the image is blurry or dark. So, I built a 4-step preprocessing pipeline using OpenCV:
1. **Grayscale**: Converted images to black-and-white.
2. **Blurring**: Removed background noise.
3. **Thresholding**: Made the text pop out against the background.
4. **Morphological Transformations**: Fixed broken letters and removed speckles.

## 💻 How to Run Locally

### 1. Install Tesseract OCR on your System
* **Windows**: Download from [UBMannheim Tesseract Installer](https://github.com/UB-Mannheim/tesseract/wiki) and add it to your PATH.
* **Linux**: `sudo apt-get install tesseract-ocr`
* **macOS**: `brew install tesseract`

### 2. Install Python Packages
```bash
pip install opencv-python pytesseract pillow numpy matplotlib
```

### 3. Run the Script
```bash
python ocr_engine.py --image sample_images/invoice_sample.jpg --output output_texts/result.txt
```
